# bambu lab lamp kit case
I created a custom lamp for myself because it's rvery dark where I sit, so I needed some lighting and I thought "why don't I design my own?" I'm really proud of the finished product and can't wait to build it irl. So I created multiple different design while experimenting with fusion and finally settled on a design featuring multiple rings and rods connecting them together, each to be printed speperately and assembled together.

More info can be found here:
